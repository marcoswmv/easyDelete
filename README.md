# Easy Delete

An iOS app that makes easy for the user to delete as many contacts as needed, as well as recovering them whenever needed in a period of 30 days.

## Features

- Light/dark mode toggle
- Fullscreen mode
- Cross platform

## Requirements

- iOS 13.0+
- Xcode 12.5

## Tech Stack

- UIKit
- Contacts(from Apple)
- Combine
- CoreData
